Audio placeholders
==================

The frontend safely attempts to load these optional files:

- background.wav
- click.wav
- ai-move.wav
- win.wav
- lose.wav
- draw.wav
- skill.wav
- skill_prediction.mp3
- skill_block.mp3
- skill_random.mp3
- skill_cadence.mp3

If the files are absent, the audio manager marks them unavailable and the game continues without errors.
