# UI Prompt - Phrolova's Tic Tac Toe

Design a dark gothic Tic Tac Toe web game where the opponent, Phrolova, feels like an elegant melancholic conductor leading a crimson-black performance.

## Visual Direction

- Theme: red-black gothic, dark fantasy, elegant, theatrical.
- Mood: mysterious, poetic, melancholic, refined, dramatic.
- Palette: deep black, charcoal, blood red, crimson glow, antique gold accents, pale bone text.
- Lighting: stage-like glow, red ambient haze, subtle cathedral shadows.
- Texture: faint grid, velvet darkness, glassy cards, thin ornamental borders.

## Phrolova Character

- Phrolova is the conductor of the match.
- She speaks formally, dramatically, and poetically.
- Her presence should feel calm, dangerous, and graceful.
- Avatar can be a placeholder portrait: pale face, dark conductor coat, crimson aura, gold baton.

## Layout

- Header as a theatrical title card.
- Main board appears like a stage or musical score.
- Phrolova avatar panel on the side.
- Score panel resembles a concert ledger.
- Dialog panel resembles a whispered monologue from a gothic opera.
- Controls are rounded but sharp in mood, with crimson gradients and gold borders.
- Player Skill panel sits below the board as a cool blue-white counterpoint to Phrolova's crimson stage.

## Board Concept

- The Tic Tac Toe board is the stage.
- Each cell is a dark tile with subtle crimson inner glow.
- X and O should feel like musical notes or marks in a score.
- Winning cells should glow like the final chord of a performance.
- Insight Move and Harmony Shield highlights should glow blue-white, clearly distinct from red victory effects.

## Interaction

- Cell hover: slight lift, crimson glow, elegant transition.
- Cell click: quick dramatic mark animation.
- Win: highlighted line/cells with gold-crimson emphasis.
- Dialog changes after moves and results.
- Skill buttons show remaining uses, disabled states, and concise tooltip-style descriptions.
- Match finale should appear as a modal curtain call.

## Typography

- Use expressive serif typography.
- Avoid sterile default UI font stacks.
- Title should feel like a gothic opera poster.
- Body text should remain readable but poetic.

## Sample Dialog Mood

- "Let the stage be set..."
- "The final note has fallen."
- "A discord... how unfortunate."
- "A suspended chord. Neither victory nor ruin dares to breathe."
- "The hall is mine. Every note returns to Phrolova."

## Responsive Behavior

- Desktop: three-column arena with avatar, board, and score/dialog panels.
- Tablet/mobile: stack panels vertically, keep board large and touch-friendly.
- Preserve the theatrical mood on small screens with compact spacing and strong contrast.
